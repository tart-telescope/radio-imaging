# Basic Radio Telescope imaging and Calibration

Radio Telescope Imaging tutorial using stimela. This material is focused on using data from the [TART radio telescope](https://tart.elec.ac.nz).

## Running these tutorials in your browser

You can run these tutorials on a cloud based linux computer accessible via github codespaces. You will need a github account to access this feature.

1. Log into github (works with a free account)
2. Click on the greed Code button, and select New Codespace in the codespace tab.
3. First step that needs to be done only once:  (do it in the default directory /workspaces/radio-imaging)

    source ./venv/bin/activate

# What's in this repository

The material in this repository includes the following

## TART imaging basics

This is a tutorial that uses tart_cargo to download some data and create images. Files for this tutorial belong in
the ```basics``` directory.

## TART calibration using CASA

This tutorial is part of a TART workshop presentation on imaging TART using CASA. Files for this tutorial belong in the ```casa``` directory
